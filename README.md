# Student Housing Management

A comprehensive smart contract system for managing student housing with integrated roommate matching, lease management, maintenance requests, and security deposit handling.

## Key Features

• **Property Management** - Add and manage dormitory and off-campus housing properties
• **Lease Management** - Create, activate, and terminate student leases with security deposit tracking
• **Maintenance Requests** - Submit and track property maintenance requests with priority levels
• **Roommate Matching** - Create profiles for roommate matching based on preferences and budget
